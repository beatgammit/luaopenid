This is an example usage of the Lua based implementation of the OpenID distributed authentication architecture. This is a trivial and insecure example, and it is provided only for illustrative purposes.

To try out this example, you should use the reference OpenID consumer located at www.danga.com. This example server currently has two configured users that can be used:

  • user: alice, passwd: apple, openid_url: http://<your URL>/alice.html
  • user: bob, passwd: banana, openid_url: http://<your URL>/bob.html
